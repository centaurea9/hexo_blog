原本的布局我个人看来过于拥挤了，总感觉看起来不是很舒心，于是对中间区域进行了调整
原本控制的代码如下：
```
@media (min-width: 1440px) {
    article {
        width: calc(100% - 520px);
    }
}
```
意思就是如果用户的屏幕宽度超过1400px时，将article区域设置为占满空间但要减去520px
于是我将520px改为了650px，这样中间就显得苗条些，不会有那种拥挤的视觉效果
修改完代码块粘贴在custom.css最下端即可
